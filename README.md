Protección e Innovación en los Humedales de Bogotá
 Descripción del proyecto
Este proyecto consiste en el desarrollo de una página web informativa enfocada en los humedales de Bogotá, con el objetivo de generar conciencia sobre su importancia ecológica, los problemas ambientales que enfrentan y la necesidad de su protección.
La página integra contenido educativo, análisis de datos, visualización interactiva y una encuesta para evaluar la percepción del usuario frente a la conservación ambiental.

 Objetivos
Informar sobre la importancia de los humedales en Bogotá
Analizar problemáticas ambientales actuales
Visualizar datos mediante gráficos interactivos
Promover la conciencia ambiental y el emprendimiento sostenible
Evaluar el nivel de conocimiento del usuario mediante una encuesta

 Estructura del proyecto
Inicio: Presentación del sitio y contexto general
Introducción: Explicación sobre los humedales
Importancia: Beneficios ecológicos (biodiversidad, agua, clima)
Problemas ambientales: Contaminación, urbanización, impacto humano
Organizaciones: Entidades que protegen los humedales
Datos y análisis: Visualización de información mediante gráficos
Encuesta: Evaluación interactiva del usuario

 Tecnologías utilizadas
HTML5
CSS3
JavaScript
Chart.js (visualización de datos)

 Funcionalidades principales
Diseño responsive y estructurado
Gráficos interactivos:
Gráfico de torta
Gráfico de barras
Gráfico de líneas
Gráfico de área
Comparador visual de imágenes (impacto ambiental)
Encuesta con almacenamiento en localStorage
Navegación mediante menú interactivo

Datos abiertos ambientales
Artículos académicos sobre humedales
Plataformas de análisis de biodiversidad

 Impacto del proyecto
Este sitio busca contribuir a:
La educación ambiental
La protección de ecosistemas urbanos
La concientización ciudadana
El uso de tecnología para el análisis ambiental
